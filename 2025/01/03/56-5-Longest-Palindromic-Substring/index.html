56-5. Longest Palindromic Substring

發表於 2025-01-03 分類於 grind75

題目描述

Given a string s, return the longest palindromic substring in s.

Example 1:

Input: s = “babad”
Output: “bab”
Explanation: “aba” is also a valid answer.
Example 2:

Input: s = “cbbd”
Output: “bb”

Constraints:

1 <= s.length <= 1000
s consist of only digits and English letters.

想法

符合palindrome的字串長度可能會是odd（奇數）或even（偶數）:

  • odd palindrome: “aba”, “c”, “abcba”
  • even palindrome: “”, “aa”, “abba”

因此我們只需要將s中的每個字符都去計算odd palindrome和even palindrome，然後順便更新結果即可。

解法

1. Two pointers

Time complexity: O(n^2)

  • 對於s中的每個字符，都要去計算由該字符的odd longest palindrome和even longest palindrome。

Space complexity: O(1)
